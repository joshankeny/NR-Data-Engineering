## 4. GitHub Foundations Certification (GH-900)

[View Certification Details](https://learn.microsoft.com/en-us/credentials/certifications/github-foundations/?WT.mc_id=certposter_poster_wwl&practice-assessment-type=certification)

### Focus:

- An **entry-level certification** from Microsoft/GitHub that validates your ability to:
	- Work with GitHub for version control, collaboration, and basic DevOps workflows
- Git + Version Control Fundamentals
	- What version control is and why it matters
	- Repositories, commits, branches
	- How Git tracks changes
- Working with Repositories
	- Creating and managing repos
	- README, LICENSE, CONTRIBUTING files
	- Branching strategies
- Collaboration in GitHub
	- Pull requests
	- Code reviews
	- Issues and discussions
	- Linking PRs to issues
- Modern Development Practices
	- GitHub Actions (CI/CD basics)
	- GitHub Copilot (AI coding)
	- Codespaces (cloud dev environments)
- Project Management in GitHub
	- Projects, labels, milestones
	- Tracking work and progress
	- Workflow organization
- Security + Access Control
	- Permissions and roles
	- Branch protection rules
	- 2FA and account security
- Open Source + Community
	- Forking vs cloning
	- Contributing to open source
	- GitHub ecosystem

### Details:

- ~100 minutes
- ~$99
- Recommended Official Learning Path
	- ~**11–12 hours total training**
	    - Part 1: ~6 hr 44 min
	    - Part 2: ~4 hr 57 min
	- Realistic Prep Time
		- Beginner: **1–2 weeks**
		- With Git experience: **2–4 days**
